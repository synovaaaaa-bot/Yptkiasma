# Logo Files

Silakan tambahkan 2 file logo berikut ke folder ini:

## Logo yang Diperlukan:

### 1. `logo-iasma-1.png` (atau .jpg, .svg)
- **Deskripsi**: Logo dengan "IASMA 1 LANDBOUW" dan "BUKITTINGGI"
- **Fitur**: 
  - Graphic biru di tengah (menyerupai bangunan/pohon)
  - Text "IASMA" di kiri atas, "1" di tengah atas, "LANDBOUW" di kanan atas
  - Text "BUKITTINGGI" di bawah
  - Green arcs di kiri dan kanan
- **Format**: PNG, JPG, atau SVG
- **Ukuran disarankan**: 200x200px atau lebih besar (akan di-resize otomatis)

### 2. `logo-ytpk.png` (atau .jpg, .svg)
- **Deskripsi**: Logo circular dengan border biru tebal
- **Fitur**:
  - Bentuk circular dengan border biru
  - Text "IASMA SATU LANDBOUW BUKITTINGGI" di bagian atas
  - Text "YAYASAN TIM PEDULI KEMANUSIAAN" di bagian bawah
  - Graphic biru di tengah dengan elemen arsitektur tradisional
- **Format**: PNG, JPG, atau SVG
- **Ukuran disarankan**: 200x200px atau lebih besar (akan di-resize otomatis)

## Cara Menambahkan:

1. Simpan file logo dengan nama yang tepat:
   - `logo-iasma-1.png` (atau .jpg/.svg)
   - `logo-ytpk.png` (atau .jpg/.svg)

2. Pastikan file berada di folder: `public/assets/logos/`

3. Logo akan otomatis muncul di header/navbar website

## Catatan:

- Jika file logo tidak ditemukan, sistem akan menggunakan placeholder SVG
- Format PNG dengan transparansi direkomendasikan untuk hasil terbaik
- Pastikan logo memiliki resolusi yang cukup untuk tampilan HD
